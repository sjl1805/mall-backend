# 用户模块VO类设计方案

## 1. UserBasicVO
作用：用户基础信息展示
字段：
- Long id
- String username
- String nickname
- String phone
- String email
- String avatar
- UserGenderEnum gender
- UserStatusEnum status
- UserRoleEnum role
- LocalDateTime createTime

## 2. UserDetailVO
作用：用户详细信息展示（扩展统计字段）
字段：
- UserBasicVO基础字段
- Integer addressCount
- Integer orderCount
- Integer couponCount
- LocalDateTime lastLoginTime

## 3. UserAddressVO
作用：收货地址信息展示
字段：
- Long id
- String receiverName
- String receiverPhone
- String fullAddress（省+市+区+详细地址）
- UserAddressStatusEnum isDefault
- LocalDateTime updateTime

## 4. UserCouponVO
作用：用户优惠券信息
字段：
- Long id
- String couponName（来自Coupon）
- CouponTypeEnum couponType
- BigDecimal value
- BigDecimal minAmount
- LocalDateTime startTime
- LocalDateTime endTime
- UserCouponStatusEnum status

## 5. UserBehaviorVO
作用：用户行为记录
字段：
- Long id
- String productName（来自Products）
- String behaviorTypeDesc（行为类型文字说明）
- LocalDateTime behaviorTime
- Integer duration
- String deviceType

## 6. FavoriteFolderVO
作用：收藏夹信息展示
字段：
- Long id
- String name
- String description
- Integer itemCount
- FavoriteFolderStatusEnum isPublic
- LocalDateTime updateTime

## 7. ProductFavoriteVO
作用：商品收藏信息
字段：
- Long id
- String productName
- BigDecimal price
- String mainImage（首张图片）
- LocalDateTime createTime

## 8. CartVO
作用：购物车条目详情
字段：
- Long id
- String productName
- String specValues（规格组合）
- BigDecimal price
- Integer quantity
- CartCheckedStatusEnum checked
- Integer stock（实时库存）

## 9. OrderPreviewVO
作用：订单预览信息
字段：
- String orderNo
- BigDecimal payAmount
- OrderStatusEnum status
- String firstProductImage
- Integer totalQuantity
- LocalDateTime createTime

## 10. OrderDetailVO
作用：订单详细信息
字段：
- OrderPreviewVO基础字段
- List<OrderItemVO> items
- String receiverInfo（姓名+电话）
- String fullAddress
- String logisticsInfo（公司+运单号）

## 11. ReviewVO
作用：商品评价信息
字段：
- Long id
- String productName
- Integer rating
- String content
- List<String> images
- String nickname（来自Users）
- LocalDateTime createTime

## 12. UserStatisticsVO
作用：用户行为统计
字段：
- Integer viewCount（浏览数）
- Integer favoriteCount（收藏数）
- Integer orderCount（订单数）
- Integer reviewCount（评价数）
- LocalDateTime lastActiveTime

## 13. UserProfileVO
作用：个人中心综合信息
字段：
- UserBasicVO基础信息
- UserStatisticsVO统计信息
- Integer unreadMessageCount
- Integer pendingOrders

## 14. UserLoginVO
作用：登录返回信息
字段：
- String token
- UserBasicVO userInfo
- List<String> permissions
- LocalDateTime loginTime

## 15. UserPageVO
作用：用户分页结果
字段：
- Integer pageNum
- Integer pageSize
- Long total
- List<UserBasicVO> list

# 设计原则说明
1. 安全脱敏：所有VO均不包含passwordHash/salt等敏感字段
2. 关联查询：通过@TableField(select = false)控制MyBatis-Plus查询
3. 枚举转换：使用@JsonFormat配合自定义序列化器转换枚举值为中文
4. 地址聚合：在VO层处理省市区字段合并为完整地址
5. 图片处理：商品图片JSON数组在VO层转换为首图URL
6. 状态映射：订单状态等业务状态添加文字说明字段
7. 分页封装：统一分页响应格式保持接口规范

# 订单模块VO类扩展设计

## 1. OrderListVO
作用：订单列表项展示
字段：
- String orderNo
- BigDecimal payAmount
- OrderStatusEnum status
- String statusDesc（状态文字说明）
- String firstProductImage
- Integer productCount
- LocalDateTime createTime（格式化为yyyy-MM-dd HH:mm）

## 2. OrderDetailVO（增强）
作用：订单完整详情展示
字段：
- OrderListVO基础字段
- List<OrderItemDetailVO> items
- String paymentMethodDesc（支付方式说明）
- String receiverInfo（收件人+电话）
- String fullAddress（完整地址）
- String logisticsInfo（物流公司+运单号）
- BigDecimal couponAmount（优惠券抵扣金额）
- String couponName（使用优惠券名称）

## 3. OrderItemDetailVO
作用：订单商品明细
字段：
- String productName
- String specValues
- BigDecimal price
- Integer quantity
- String productImage
- BigDecimal itemTotal
- ReviewStatusEnum reviewStatus（评价状态）

## 4. OrderCreateVO
作用：订单创建返回信息
字段：
- String orderNo
- BigDecimal payAmount
- LocalDateTime expireTime（支付截止时间）
- List<Long> couponIds（适用的优惠券ID列表）

## 5. OrderPaymentVO
作用：支付信息返回
字段：
- String orderNo
- BigDecimal payAmount
- String paymentForm（支付表单内容）
- String qrCodeUrl（支付二维码URL）
- Integer remainSeconds（剩余支付秒数）

## 6. OrderLogisticsVO
作用：物流信息展示
字段：
- String logisticsCompany
- String trackingNumber
- String latestStatus
- LocalDateTime updateTime
- List<LogisticsTrace> traces（物流轨迹）

## 7. OrderStatisticsVO
作用：订单数据统计
字段：
- Integer pendingPayment（待支付）
- Integer pendingDelivery（待发货）
- Integer pendingReceive（待收货）
- BigDecimal monthlyAmount（本月消费金额）
- Integer returnCount（售后数量）

## 8. OrderPageVO
作用：订单分页结果
字段：
- Integer pageNum
- Integer pageSize
- Long total
- List<OrderListVO> list

# 设计增强说明
1. 状态增强：所有状态枚举字段增加对应的文字说明字段（xxxDesc）
2. 时间格式化：使用@JsonFormat(pattern = "yyyy-MM-dd HH:mm")统一时间格式
3. 金额处理：BigDecimal字段使用自定义序列化器保留两位小数
4. 支付安全：支付相关VO不包含敏感支付信息，通过支付平台接口获取
5. 物流轨迹：LogisticsTrace内部类包含时间+地点+描述信息
6. 商品快照：保留订单创建时的商品信息，避免后续商品信息变更影响订单数据

## 4. ProductSpecVO

- String specName

## 5. ReviewStatisticsVO

# 设计要点说明
1. 价格展示：商品列表显示最低SKU价格，详情页展示完整SKU价格体系
2. 图片处理：使用FastJSON的JSONArray转换图片JSON字符串为List<String>
3. 库存状态：根据库存数量显示「充足」「紧张」「缺货」等状态
4. 规格组合：使用Map存储规格键值对，前端渲染时生成规格矩阵
5. 评价统计：计算好评率、带图评价比例等数据
6. 分类展示：使用树形结构VO实现三级分类联动
7. 实时价格：商品详情价格取自SKU表，与主商品表解耦
8. 推荐算法：展示推荐权重和算法版本便于后期优化
9. 收藏状态：关联收藏夹信息展示多维数据
10. 缓存策略：为高频访问的VO数据添加@Cacheable注解
